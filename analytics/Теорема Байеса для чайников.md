–§—Ä–∞–≥–º–µ–Ω—Ç –∏–∑ –∫/—Å "–¢–µ–æ—Ä–∏—è –±–æ–ª—å—à–æ–≥–æ –≤–∑—Ä—ã–≤–∞"

**P(A|B)**¬†–æ–∑–Ω–∞—á–∞–µ—Ç —É—Å–ª–æ–≤–Ω—É—é –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è A, –µ—Å–ª–∏ –ø—Ä–æ–∏–∑–æ—à–ª–æ —Å–æ–±—ã—Ç–∏–µ B.¬†**P(A)**¬†- –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è A (–≥–∏–ø–æ—Ç–µ–∑–∞).¬†**P(B)**¬†- –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è B.

–§–æ—Ä–º—É–ª–∞ –ë–∞–π–µ—Å–∞

–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Å–æ–±—ã—Ç–∏—è –º–æ–∂–µ—Ç –±—ã—Ç—å –æ—Ç 0% –¥–æ 100% (–∏–ª–∏ –æ—Ç 0 –¥–æ 1). –ï—Å–ª–∏ –æ–±–æ–∑–Ω–∞—á–∏–º —Å–æ–±—ã—Ç–∏–µ –±—É–∫–≤–æ–π¬†**A**, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —ç—Ç–æ–≥–æ —Å–æ–±—ã—Ç–∏—è (A) –±—É–¥–µ—Ç¬†**P(A)**. –ù–∞–ø—Ä–∏–º–µ—Ä, –Ω–∞ —à–µ—Å—Ç–∏–≥—Ä–∞–Ω–Ω–æ–º –∫—É–±–∏–∫–µ üé≤ 6 –≥—Ä–∞–Ω–µ–π –∏ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã–ø–∞–¥–µ–Ω–∏—è –ª—é–±–æ–≥–æ —á–∏—Å–ª–∞ –æ–¥–∏–Ω–∞–∫–æ–≤–∞, –ø–æ—ç—Ç–æ–º—É –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –≤—ã–ø–∞–¥–µ–Ω–∏—è –ª—é–±–æ–≥–æ —á–∏—Å–ª–∞ –±—É–¥–µ—Ç 1/6 (–∏–ª–∏ 0.166..; –∏–ª–∏ 16,66..%).¬†**P(1)**=**P(2)**..=**P(6)**=16.66..%. –°—É–º–º–∞ –∂–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–µ–π –≤—Å–µ—Ö (–Ω–µ—Å–æ–≤–º–µ—Å—Ç–∏–º—ã—Ö) —Å–æ–±—ã—Ç–∏–π –±—É–¥–µ—Ç 100% (–∏–ª–∏ 1).

## –ù–∞ –ø—Ä–∏–º–µ—Ä–µ –∫–æ—Ä–æ–±–æ–∫ —Å —à–∞—Ä–∏–∫–∞–º–∏

–î–æ–ø—É—Å—Ç–∏–º, —É –Ω–∞—Å –µ—Å—Ç—å –¥–≤–µ –∫–æ—Ä–æ–±–∫–∏: –∂–µ–ª—Ç–∞—è –∏ –∑–µ–ª–µ–Ω–∞—è. –í –∫–∞–∂–¥–æ–π –∫–æ—Ä–æ–±–∫–µ –ª–µ–∂–∞—Ç —à–∞—Ä–∏–∫–∏: –∫—Ä–∞—Å–Ω—ã–µ –∏ —Å–∏–Ω–∏–µ. –í –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–µ 1 —Å–∏–Ω–∏–π —à–∞—Ä–∏–∫ –∏ 1 –∫—Ä–∞—Å–Ω—ã–π, –≤ –∑–µ–ª–µ–Ω–æ–π - 6 —Å–∏–Ω–∏—Ö –∏ 2 –∫—Ä–∞—Å–Ω—ã—Ö. –ï—Å–ª–∏ –º—ã –≤—Å–ª–µ–ø—É—é –¥–æ—Å—Ç–∞–ª–∏ –∫—Ä–∞—Å–Ω—ã–π —à–∞—Ä–∏–∫, —Ç–æ –∏–∑ –∫–∞–∫–æ–π –∫–æ—Ä–æ–±–∫–∏ –æ–Ω –≤–µ—Ä–æ—è—Ç–Ω–µ–µ –≤—Å–µ–≥–æ?

–î–≤–µ –∫–æ—Ä–æ–±–∫–∏ —Å —à–∞—Ä–∏–∫–∞–º–∏

–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å (–≤—Å–ª–µ–ø—É—é) –¥–æ—Å—Ç–∞—Ç—å —Å–∏–Ω–∏–π —à–∞—Ä–∏–∫ (**C**) –∏–∑ –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–∏ (**–ñ**) –±—É–¥–µ—Ç¬†**P(–°|–ñ)**¬†= 50%. –†–∞—Å—Å—á–∏—Ç—ã–≤–∞–µ—Ç—Å—è –∫–∞–∫ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–∏–Ω–∏—Ö —à–∞—Ä–∏–∫–æ–≤ –≤ –∫–æ—Ä–æ–±–∫–µ, –¥–µ–ª–µ–Ω–Ω–æ–µ –Ω–∞ –æ–±—â–µ–µ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —à–∞—Ä–∏–∫–æ–≤ –≤ –∫–æ—Ä–æ–±–∫–µ. 1/2 = 0.5 –∏–ª–∏ 50%. –ê–Ω–∞–ª–æ–≥–∏—á–Ω–æ –¥–ª—è –∫—Ä–∞—Å–Ω—ã—Ö —à–∞—Ä–∏–∫–æ–≤ (**–ö**) –∏ –∑–µ–ª–µ–Ω–æ–π (**–ó**) –∫–æ—Ä–æ–±–∫–∏:¬†**P(–ö|–ñ)**¬†= 1/(1+1) = 50%;¬†**P(C|–ó)**¬†= 6/(6+2) = 75%;¬†**P(–ö|–ó)**¬†= 2/(6+2) = 25%.

–£—Å–ª–æ–≤–Ω—ã–µ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç–∏ —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç –∫–æ—Ä–æ–±–∫–∏

–ï—Å–ª–∏ –º—ã –¥–æ—Å—Ç–∞–Ω–µ–º —Å–ª—É—á–∞–π–Ω—ã–π —à–∞—Ä–∏–∫, —Ç–æ –≤ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞ –º—ã –º–æ–∂–µ–º –≤—ã—á–∏—Å–ª–∏—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, –∏–∑ –∫–∞–∫–æ–π –∫–æ—Ä–æ–±–∫–∏ –º—ã –µ–≥–æ –¥–æ—Å—Ç–∞–ª–∏ (–ø–æ —Ñ–æ—Ä–º—É–ª–µ –ë–∞–π–µ—Å–∞).¬†**P(–ñ|–°)**¬†- –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–∏ –ø—Ä–∏ —Å–∏–Ω–µ–º —à–∞—Ä–∏–∫–µ;¬†**P(–ñ|–ö)**¬†- –∂–µ–ª—Ç–æ–π –ø—Ä–∏ –∫—Ä–∞—Å–Ω–æ–º –∏ —Ç.–¥.

![–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –æ—Ç —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞](https://habrastorage.org/r/w1560/getpro/habr/upload_files/877/a90/82c/877a9082c99ae0f10438bda221afc3f7.png "–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –æ—Ç —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞")

–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –æ—Ç —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞

–í—Å–µ–≥–æ —à–∞—Ä–∏–∫–æ–≤ –≤ –æ–±–µ–∏—Ö –∫–æ—Ä–æ–±–∫–∞—Ö (**N**) = 1+1+6+2=10. –í—Å–µ–≥–æ –≤ –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–µ (**–ñ**) = 1+1=2 —à–∞—Ä–∏–∫–∞, –≤ –∑–µ–ª–µ–Ω–æ–π (**–ó**) = 2+6=8 —à–∞—Ä–∏–∫–æ–≤. –í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ —Å–ª—É—á–∞–π–Ω—ã–π —à–∞—Ä–∏–∫ –±—É–¥–µ—Ç –∏–∑ –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–∏ –±—É–¥–µ—Ç¬†**P(–ñ)**¬†= 20%; –∏–∑ –∑–µ–ª–µ–Ω–æ–π¬†**P(–ó)**¬†= 80%.

![–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –≤—Å–µ—Ö —à–∞—Ä–∏–∫–æ–≤](https://habrastorage.org/r/w1560/getpro/habr/upload_files/627/8f0/b76/6278f0b76de71e01a0e1b4950299c4ad.png "–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –≤—Å–µ—Ö —à–∞—Ä–∏–∫–æ–≤")

–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ –∫–æ—Ä–æ–±–∫–∏ –≤—Å–µ—Ö —à–∞—Ä–∏–∫–æ–≤

–í—Å–µ–≥–æ –∫—Ä–∞—Å–Ω—ã—Ö —à–∞—Ä–∏–∫–æ–≤ (**–ö**) = 1+2=3, —Å–∏–Ω–∏—Ö (**–°**) = 1+6=7. –ï—Å–ª–∏ –º—ã –¥–æ—Å—Ç–∞–µ–º —à–∞—Ä–∏–∫ –∏–∑ –æ–±—â–µ–π –∫—É—á–∏ (—Ç.–µ. –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç –∫–æ—Ä–æ–±–∫–∏), —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–≥–æ, —á—Ç–æ –æ–Ω –±—É–¥–µ—Ç –∫—Ä–∞—Å–Ω—ã–º,¬†**P(–ö)**=30%; —Å–∏–Ω–∏–º¬†**P(–°)**=70%.

![–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞ –∏–∑ –æ–±—â–µ–≥–æ —á–∏—Å–ª–∞](https://habrastorage.org/r/w1560/getpro/habr/upload_files/041/d02/3f1/041d023f1d08687651e68b7214e27cf0.png "–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞ –∏–∑ –æ–±—â–µ–≥–æ —á–∏—Å–ª–∞")

–í–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ü–≤–µ—Ç–∞ —à–∞—Ä–∏–∫–∞ –∏–∑ –æ–±—â–µ–≥–æ —á–∏—Å–ª–∞

–ï—Å–ª–∏ –º—ã –Ω–µ –≤–∏–¥–µ–ª–∏, –∏–∑ –∫–∞–∫–æ–π –∫–æ—Ä–æ–±–∫–∏ –¥–æ—Å—Ç–∞–ª–∏ —à–∞—Ä–∏–∫, —Ç–æ –ø–æ —Ü–≤–µ—Ç—É —à–∞—Ä–∏–∫–∞ –º—ã –º–æ–∂–µ–º –æ–ø—Ä–µ–¥–µ–ª–∏—Ç—å –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å —Ç–æ–π –∏–ª–∏ –∏–Ω–æ–π –∫–æ—Ä–æ–±–∫–∏ –ø–æ —Ñ–æ—Ä–º—É–ª–µ –ë–∞–π–µ—Å–∞. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ –∫—Ä–∞—Å–Ω—ã–π —à–∞—Ä–∏–∫ –±—ã–ª –∏–∑ –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–∏.

**P(–ñ|–ö)**¬†= P(–ö|–ñ) * P(–ñ) / P(–ö) = 0.5 * 0.2 / 0.3 = 0.333.. = 33.3% =¬†**1/3**. –ó–Ω–∞—á–∏—Ç, –µ—Å–ª–∏ –º—ã –¥–æ—Å—Ç–∞–Ω–µ–º –∫—Ä–∞—Å–Ω—ã–π —à–∞—Ä–∏–∫, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ –æ–Ω –∏–∑ –∂–µ–ª—Ç–æ–π –∫–æ—Ä–æ–±–∫–∏ - 1/3.

–¢.–∫. –≤ —Å—É–º–º–µ –†(–ñ|–ö)+–†(–ó|–ö) = 1, —Ç.–∫. –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ –∫—Ä–∞—Å–Ω—ã–π —à–∞—Ä–∏–∫ –ª–∏–±–æ –∏–∑ –∫—Ä–∞—Å–Ω–æ–π –ª–∏–±–æ –∏–∑ –∑–µ–ª–µ–Ω–æ–π –∫–æ—Ä–æ–±–∫–∏ 100%, —Ç–æ –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å –∫—Ä–∞—Å–Ω–æ–≥–æ —à–∞—Ä–∏–∫–∞ –∏–∑ –∑–µ–ª–µ–Ω–æ–π –∫–æ—Ä–æ–±–∫–∏:

**P(–ó|–ö)**¬†= 1 - –†(–ñ|–ö) = 1 - 1/3 =¬†**2/3**

–ò–ª–∏ –ø–æ —Ñ–æ—Ä–º—É–ª–µ: P(–ó|–ö) = P(–ö|–ó) * P(–ó) / P(–ö) = 0.25 * 0.8 / 0.3 = 0.666.. = 66.6% = 2/3

–î–ª—è —Å–∏–Ω–µ–≥–æ —à–∞—Ä–∏–∫–∞ –∏–∑ –∂–µ–ª—Ç–æ–π –∏ –∑–µ–ª–µ–Ω–æ–π –∫–æ—Ä–æ–±–∫–∏:

**P(–ñ|–°)**¬†= P(–°|–ñ) * P(–ñ) / P(–°) = 0.5 * 0.2 / 0.7 = 0.0,1428.. = 14.28% =¬†**1/7**

**P(–ó|–°)**¬†= 1 - 1/7 =¬†**6/7**

–ò–ª–∏ –ø–æ —Ñ–æ—Ä–º—É–ª–µ: P(–ó|–°) = P(–°|–ó) * P(–ó) / P(–°) = 0,75 * 0,8 / 0,7 = 0,8571 = 85.71..% = 6/7
