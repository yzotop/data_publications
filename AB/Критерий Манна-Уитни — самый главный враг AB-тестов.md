–≠—Ç–æ—Ç –∫—Ä–∏—Ç–µ—Ä–∏–π –æ—á–µ–Ω—å –ø–æ–ø—É–ª—è—Ä–µ–Ω. –í–æ –º–Ω–æ–≥–∏—Ö –∫–æ–º–ø–∞–Ω–∏—è—Ö –∏ –Ω–∞ –æ–±—É—á–∞—é—â–∏—Ö –∫—É—Ä—Å–∞—Ö —Ä–∞—Å—Å–∫–∞–∑—ã–≤–∞—é—Ç –ø—Ä–æ –¥–≤–∞ –µ–≥–æ –≤–∞–∂–Ω—ã—Ö –ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–∞:

2. –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ ‚Äî —Ä–æ–±–∞—Å—Ç–Ω—ã–π –∫—Ä–∏—Ç–µ—Ä–∏–π. –í –¥–∞–Ω–Ω—ã—Ö —á–∞—Å—Ç–æ –±—ã–≤–∞–µ—Ç –º–Ω–æ–≥–æ —à—É–º–æ–≤ –∏ –≤—ã–±—Ä–æ—Å–æ–≤, –ø–æ—ç—Ç–æ–º—É T-test –Ω–µ–ø—Ä–∏–º–µ–Ω–∏–º –ø–æ —Å–æ–æ–±—Ä–∞–∂–µ–Ω–∏—è–º –º–æ—â–Ω–æ—Å—Ç–∏, —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∏–ª–∏ –Ω–µ–Ω–æ—Ä–º–∞–ª—å–Ω–æ—Å—Ç–∏ –¥–∞–Ω–Ω—ã—Ö. –ê –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ –≤ —ç—Ç–æ—Ç –º–æ–º–µ–Ω—Ç –æ—Ç–ª–∏—á–Ω–æ —Å—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç –∏ –±–æ–ª–µ–µ –≤–µ—Ä–æ—è—Ç–Ω–æ –Ω–∞—Ö–æ–¥–∏—Ç —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–∏ –∑–Ω–∞—á–∏–º—ã–π —ç—Ñ—Ñ–µ–∫—Ç.
    

–ï—Å–ª–∏ –≤—ã –∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞–ª–∏ A/B-—Ç–µ—Å—Ç, –≥–¥–µ –≤–∞—Å –∏–Ω—Ç–µ—Ä–µ—Å–æ–≤–∞–ª –ø—Ä–∏—Ä–æ—Å—Ç –∏–ª–∏ –ø–∞–¥–µ–Ω–∏–µ –∫–∞–∫–æ–π-—Ç–æ –º–µ—Ç—Ä–∏–∫–∏, —Ç–æ –Ω–∞–≤–µ—Ä–Ω—è–∫–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª–∏ –∫—Ä–∏—Ç–µ—Ä–∏–π –ú–∞–Ω–Ω–∞-–£–∏—Ç–Ω–∏. –Ø —Ö–æ—á—É —Ä–∞—Å—Å–∫–∞–∑–∞—Ç—å –ø—Ä–æ –ø–æ–¥–≤–æ–¥–Ω—ã–µ –∫–∞–º–Ω–∏ —ç—Ç–æ–≥–æ –∫—Ä–∏—Ç–µ—Ä–∏—è, –∏ –ø–æ—á–µ–º—É –º—ã –≤ –∫–æ–º–ø–∞–Ω–∏–∏ –µ–≥–æ –Ω–µ –∏—Å–ø–æ–ª—å–∑—É–µ–º. –ê –≤ –∫–æ–Ω—Ü–µ –≤—ã –ø–æ–π–º—ë—Ç–µ, –æ—Ç–∫—É–¥–∞ –≤–∑—è–ª—Å—è —Ç–∞–∫–æ–π —Ö–æ–ª–∏–≤–∞—Ä–Ω—ã–π –∑–∞–≥–æ–ª–æ–≤–æ–∫ :)

**–ü–ª–∞–Ω —Å—Ç–∞—Ç—å–∏ —Ç–∞–∫–æ–π:**

1. –Ø —Ç–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∏ –ø–æ–∫–∞–∂—É, —á—Ç–æ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ –∏ –ø–æ—á–µ–º—É —ç—Ç–æ –Ω–µ –∏–º–µ–µ—Ç –Ω–∏—á–µ–≥–æ –æ–±—â–µ–≥–æ —Å —Ä–æ—Å—Ç–æ–º –º–µ–¥–∏–∞–Ω –∏ —Å—Ä–µ–¥–Ω–µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è.¬†**–ê –µ—â—ë —Ä–∞–∑–≤–µ—é –º–∏—Ñ, —á—Ç–æ –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç —ç—Ç—É –≥–∏–ø–æ—Ç–µ–∑—É:**
    

![H‚ÇÄ: P(T > C) = 1/2](https://habrastorage.org/getpro/habr/upload_files/c77/b3f/506/c77b3f506448766ec801afc17588999d.svg)

2. –ù–∞ –ø—Ä–∏–º–µ—Ä–µ –∏—Å–∫—É—Å—Å—Ç–≤–µ–Ω–Ω—ã—Ö –∏ —Ä–µ–∞–ª—å–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö –ø—Ä–æ–¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É—é, —á—Ç–æ –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ —Ä–∞–±–æ—Ç–∞–µ—Ç –Ω–µ —Ç–∞–∫, –∫–∞–∫ –≤—ã –æ–∂–∏–¥–∞–µ—Ç–µ. –ï–≥–æ ¬´–ø—Ä–µ–∏–º—É—â–µ—Å—Ç–≤–æ¬ª –≤ –±–æ–ª—å—à–µ–º —á–∏—Å–ª–µ –ø—Ä–æ–∫—Ä–∞—Å–æ–≤ –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ T-test, –∏ —ç—Ç–æ ‚Äî –≥–ª–∞–≤–Ω—ã–π –º–∏–Ω—É—Å —ç—Ç–æ–≥–æ –∫—Ä–∏—Ç–µ—Ä–∏—è, –ø–æ—Å–∫–æ–ª—å–∫—É –ø—Ä–æ–∫—Ä–∞—Å—ã –ª–æ–∂–Ω—ã–µ.
    
3. –†–∞—Å—Å–∫–∞–∂—É –ø—Ä–æ –ª–æ–≥–∞—Ä–∏—Ñ–º–∏—Ä–æ–≤–∞–Ω–∏–µ –º–µ—Ç—Ä–∏–∫–∏ –∏ –ø—Ä–æ–¥–µ–º–æ–Ω—Å—Ç—Ä–∏—Ä—É—é, –ø–æ—á–µ–º—É —ç—Ç–æ –ø–ª–æ—Ö–∞—è –∏–¥–µ—è.
    
4. –ü–æ–∫–∞–∂—É —Ç–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∏–µ —Å–ª—É—á–∞–∏, –∫–æ–≥–¥–∞ –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏ –ø—Ä–∏–º–µ–Ω–∏–º –∏ –µ–≥–æ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è —Å—Ä–∞–≤–Ω–µ–Ω–∏—è —Å—Ä–µ–¥–Ω–∏—Ö.
    

–û—Ç–≤–µ—á—É –Ω–∞ –≤–æ–ø—Ä–æ—Å: –∫–∞–∫ —Ç–æ–≥–¥–∞ –∂–∏—Ç—å –∏ –∫–∞–∫–æ–π –∫—Ä–∏—Ç–µ—Ä–∏–π –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞ A/B-—Ç–µ—Å—Ç–æ–≤.

### –ß—Ç–æ –ø—Ä–æ–≤–µ—Ä—è–µ—Ç –∫—Ä–∏—Ç–µ—Ä–∏–π –ú–∞–Ω–Ω–∞-–£–∏—Ç–Ω–∏?

–ü—Ä–µ–¥–ª–∞–≥–∞—é –æ—Ç–≤–µ—Ç–∏—Ç—å –Ω–∞ –≤–æ–ø—Ä–æ—Å:¬†**–º–æ–∂–µ–º –ª–∏ –º—ã –ø—Ä–æ–≤–µ—Ä—è—Ç—å —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ —Å—Ä–µ–¥–Ω–∏—Ö –∏–ª–∏ –º–µ–¥–∏–∞–Ω –≤ —Ç–µ—Å—Ç–µ –∏ –≤ –∫–æ–Ω—Ç—Ä–æ–ª–µ —Å –ø–æ–º–æ—â—å—é –∫—Ä–∏—Ç–µ—Ä–∏—è –ú–∞–Ω–Ω–∞-–£–∏—Ç–Ω–∏?**¬†–ò–ª–∏ –ø—Ä–æ–≤–µ—Ä—è—Ç—å, —á—Ç–æ –æ–¥–Ω–∞ –≤—ã–±–æ—Ä–∫–∞ ¬´–Ω–µ —Å–¥–≤–∏–Ω—É—Ç–∞¬ª –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ –¥—Ä—É–≥–æ–π –≤—ã–±–æ—Ä–∫–∏? –ò–ª–∏, –µ—Å–ª–∏ –Ω–∞ —è–∑—ã–∫–µ –º–∞—Ç–µ–º–∞—Ç–∏–∫–∏:

![H‚ÇÄ: P(T > C) = 1/2](https://habrastorage.org/getpro/habr/upload_files/497/c4f/b46/497c4fb4626a0fd50652ab7a413cc674.svg)

–ù–∞–ø—Ä–∏–º–µ—Ä, —É –Ω–∞—Å –≤ A/B-—Ç–µ—Å—Ç–µ –¥–≤–µ –≤—ã–±–æ—Ä–∫–∏:

- U[‚àí1, 1] ‚Äî —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –æ—Ç ‚àí1 –¥–æ 1
    
- U[‚àí100, 100] ‚Äî —Ä–∞–≤–Ω–æ–º–µ—Ä–Ω–æ–µ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ –æ—Ç ‚àí100 –¥–æ 100
    

–ü—Ä–æ —ç—Ç–∏ –¥–≤–∞ —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –º—ã –∑–Ω–∞–µ–º, —á—Ç–æ —É –Ω–∏—Ö —Ä–∞–≤–Ω—ã —Å—Ä–µ–¥–Ω–∏–µ –∏ –º–µ–¥–∏–∞–Ω—ã, –∏ —á—Ç–æ –æ–Ω–∏ —Å–∏–º–º–µ—Ç—Ä–∏—á–Ω—ã –æ—Ç–Ω–æ—Å–∏—Ç–µ–ª—å–Ω–æ 0. –ö—Ä–æ–º–µ —Ç–æ–≥–æ, –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å, —á—Ç–æ —Å–≥–µ–Ω–µ—Ä–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –≤ –ø–µ—Ä–≤–æ–π –≤—ã–±–æ—Ä–∫–µ –±—É–¥–µ—Ç –±–æ–ª—å—à–µ –∑–Ω–∞—á–µ–Ω–∏—è –≤–æ –≤—Ç–æ—Ä–æ–π –≤—ã–±–æ—Ä–∫–µ, —Ä–∞–≤–Ω–∞ 1/2. –ò–ª–∏, –µ—Å–ª–∏ —Å—Ñ–æ—Ä–º—É–ª–∏—Ä–æ–≤–∞—Ç—å –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏, P(T > C) = 1/2, –≥–¥–µ T –∏ C ‚Äî –≤—ã–±–æ—Ä–∫–∏ —Ç–µ—Å—Ç–∞ –∏ –∫–æ–Ω—Ç—Ä–æ–ª—è.

–Ø —Ö–æ—á—É –ø—Ä–æ–≤–µ—Ä–∏—Ç—å, —Ä–∞–±–æ—Ç–∞–µ—Ç –ª–∏ –∑–¥–µ—Å—å –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ –ú–∞–Ω–Ω-–£–∏—Ç–Ω–∏. –î–ª—è —ç—Ç–æ–≥–æ –∑–∞–ø—É—Å—Ç–∏–º —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç 1000 —Ä–∞–∑ –∏ –ø–æ—Å–º–æ—Ç—Ä–∏–º –Ω–∞ –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –æ—Ç–≤–µ—Ä–∂–µ–Ω–∏–π –Ω—É–ª–µ–≤–æ–π –≥–∏–ø–æ—Ç–µ–∑—ã —É –∫—Ä–∏—Ç–µ—Ä–∏—è. –ü–æ–¥—Ä–æ–±–Ω–µ–µ –ø—Ä–æ —ç—Ç–æ—Ç –º–µ—Ç–æ–¥ –º–æ–∂–Ω–æ –ø—Ä–æ—á–µ—Å—Ç—å –≤ –º–æ–µ–π¬†[—Å—Ç–∞—Ç—å–µ –ø—Ä–æ —É–ª—É—á—à–µ–Ω–∏–µ A/B —Ç–µ—Å—Ç–æ–≤](https://habr.com/ru/company/avito/blog/571094/). –ï—Å–ª–∏ –±—ã –∫—Ä–∏—Ç–µ—Ä–∏–π –ú–∞–Ω–Ω–∞-–£–∏—Ç–Ω–∏ —Ä–∞–±–æ—Ç–∞–ª –∫–∞–∫ –Ω–∞–¥–æ –∏ –ø—Ä–æ–≤–µ—Ä—è–ª –æ–¥–Ω—É –∏–∑ –≥–∏–ø–æ—Ç–µ–∑, —Ç–æ –ø–æ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—é¬†[—É—Ä–æ–≤–Ω—è –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏](http://www.machinelearning.ru/wiki/index.php?title=%D0%A3%D1%80%D0%BE%D0%B2%D0%B5%D0%BD%D1%8C_%D0%B7%D0%BD%D0%B0%D1%87%D0%B8%D0%BC%D0%BE%D1%81%D1%82%D0%B8)¬†–æ–Ω –±—ã –æ—à–∏–±–∞–ª—Å—è –Ω–∞ —ç—Ç–∏—Ö —Ç–µ—Å—Ç–∞—Ö –≤ 5% —Å–ª—É—á–∞–µ–≤, —Å –Ω–µ–∫–æ—Ç–æ—Ä—ã–º —Ä–∞–∑–±—Ä–æ—Å–æ–º –∏–∑-–∑–∞ —à—É–º–∞.¬†¬†

–ü—Ä–æ–≤–µ—Ä–∏–º –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ—Å—Ç—å –ø—Ä–∏–º–µ—Ä–∞ –Ω–∞ —É—Ä–æ–≤–Ω–µ –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏ –¥–ª—è T-test –∫—Ä–∏—Ç–µ—Ä–∏—è:

```
# –ü–æ–¥–∫–ª—é—á–∏–º –±–∏–±–ª–∏–æ—Ç–µ–∫–∏import scipy.stats as spsfrom tqdm.notebook import tqdm # tqdm ‚Äì –±–∏–±–ª–∏–æ—Ç–µ–∫–∞ –¥–ª—è –≤–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏–∏ –ø—Ä–æ–≥—Ä–µ—Å—Å–∞ –≤ —Ü–∏–∫–ª–µfrom statsmodels.stats.proportion import proportion_confintimport numpy as np # –ó–∞–≤–æ–¥–∏–º —Å—á–µ—Ç—á–∏–∫–∏ –∫–æ–ª–∏—á–µ—Å—Ç–≤–∞ –æ—Ç–≤–µ—Ä–≥–Ω—É—Ç—ã—Ö –≥–∏–ø–æ—Ç–µ–∑ –¥–ª—è –ú–∞–Ω–Ω–∞-–£–∏—Ç–Ω–∏ –∏ –¥–ª—è t-testmann_bad_cnt = 0ttest_bad_cnt = 0 # –ü—Ä–æ–≥–æ–Ω—è–µ–º –∫—Ä–∏—Ç–µ—Ä–∏–∏ 1000 —Ä–∞–∑sz = 1000for i in tqdm(range(sz)):    # –ì–µ–Ω–µ—Ä–∏—Ä—É–µ–º —Ä–∞—Å–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–µ    test = sps.uniform(loc=-1, scale=2).rvs(1000) # U[-1, 1]    control = sps.uniform(loc=-100, scale=200).rvs(1000) # U[-100, 100]         # –°—á–∏—Ç–∞–µ–º pvalue    mann_pvalue = sps.mannwhitneyu(control, test, alternative='two-sided').pvalue    ttest_pvalue = sps.ttest_ind(control, test, alternative='two-sided').pvalue         # –æ—Ç–≤–µ—Ä–≥–∞–µ–º –∫—Ä–∏—Ç–µ—Ä–∏–π –Ω–∞ —É—Ä–æ–≤–Ω–µ 5%    if mann_pvalue < 0.05:        mann_bad_cnt += 1     if ttest_pvalue < 0.05:        ttest_bad_cnt += 1 # –°—Ç—Ä–æ–∏–º –¥–æ–≤–µ—Ä–∏—Ç–µ–ª—å–Ω—ã–π –∏–Ω—Ç–µ—Ä–≤–∞–ª –¥–ª—è —É—Ä–æ–≤–Ω—è –∑–Ω–∞—á–∏–º–æ—Å—Ç–∏ –∫—Ä–∏—Ç–µ—Ä–∏—è (–∏–ª–∏ –¥–ª—è FPR –∫—Ä–∏—Ç–µ—Ä–∏—è)left_mann_level, right_mann_level = proportion_confint(count = mann_bad_cnt, nobs = sz, alpha=0.05, method='wilson')left_ttest_level, right_ttest_level = proportion_confint(count = ttest_bad_cnt, nobs = sz, alpha=0.05, method='wilson')# –í—ã–≤–æ–¥–∏–º —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ãprint(f"Mann-whitneyu significance level: {round(mann_bad_cnt / sz, 4)}, [{round(left_mann_level, 4)}, {round(right_mann_level, 4)}]")print(f"T-test significance level: {round(ttest_bad_cnt / sz, 4)}, [{round(left_ttest_level, 4)}, {round(right_ttest_level, 4)}]")
```

T-test –∑–¥–µ—Å—å –∫–æ—Ä—Ä–µ–∫—Ç–Ω–æ —Ä–∞–±–æ—Ç–∞–µ—Ç: –æ–Ω –æ—à–∏–±–∞–µ—Ç—Å—è –≤ 5% —Å–ª—É—á–∞–µ–≤, –∫–∞–∫ –∏ –∑–∞—è–≤–ª–µ–Ω–æ. –ê –∑–Ω–∞—á–∏—Ç, –ø—Ä–∏–º–µ—Ä –≤–∞–ª–∏–¥–µ–Ω.
